# nimind-social-assets

Public image hosting for NiMind Social posts.

Images are committed here automatically by `tools/upload_to_github_assets.py`
and served via raw.githubusercontent.com for Instagram/reel ingestion.

URL pattern: `https://raw.githubusercontent.com/ranazia517/nimind-social-assets/main/images/{filename}`

Do not manually edit this repository — it is managed by automation.
